                                function reverseString(userString){

                                    let revString = [];
                                    
                                    //reverse a string using a for loop
                                    //last position in array = array.length - 1
                                    for (let index = userString.length - 1; index >= 0 ; index--) {
                                        
                                        revString += userString[index];
                                        
                                    }

                                    return revString;

                                }

The Code is structured in three functions.

getValue()

On line 5, we hook onto the alert using getElementById() and add the Bootstrap class "invisible" so that the alert is not displayed by default. Line 7 sets the value of the user input to "userString" so we can reference it later. Line 9 saves the result of our "reverseString()" function into the variable "revString". Lines 11 through 18 performs some simple validation to make sure the user does not submit an empty string.

reverseString()

This function accepts a string as a parameter. We then set the variable "revString" to an empty array on line 26 which we will referenced later in the code. On lines 30 through 34 we iterate through our string array using a for-loop, decrementing by one starting from the last index of the string array until the index is 0. For each iteration, our "revString" variable is updated which gives us our string in reverse. The final output of the function returns the "revString" variable.

displayString()

This function takes a string (our reversed string) as a parameter. On line 45 we hook into our HTML tag with the id of "msg" and use a template literal to display the reversed string passed into the function. On line 48 we hook into our HTML tag with the id of "alert" to remove the "invisible" Bootstrap class which will make the alert visible in the browser.
